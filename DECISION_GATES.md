# Decision Gates

Use this file for consequential open questions that block specific actions.

A decision gate should not stop the whole project unless the question truly blocks the whole project. Continue unrelated work.

## Template

```markdown
### DG-YYYYMMDD-N — Short question

Status: open | answered | superseded | blocked  
Blocks: specific action(s), not the whole project unless truly global  
Safe assumption: yes/no + assumption if applicable  
Question: ...  
Evidence checked: ...  
Answer: ...  
Updated: YYYY-MM-DD
```
